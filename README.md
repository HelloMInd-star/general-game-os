## 📐 底层逻辑

### 画圈均衡范式

```text
半径 → 供给 · 速度 → 需求 · 方向 → 政策
每一次扰动 = 系统重调
静态圈 = 动态圈的特例（速度 = 0）
```

### 凯利公式 · 量化引擎

```text
f* = (b × p − q) / b
```

- **f***：最优投入比例
- **b**：净赔率
- **p**：胜率
- **q**：1 − p

### 二八管控 · 人机协作

| 状态 | 自动化 | 人工干预 | 说明 |
| :--- | :--- | :--- | :--- |
| 🟢 安全 | 80% | 20% | 系统自动运行 |
| 🟡 临界 | 60% | 40% | 建议人工复核 |
| 🔴 高风险 | 40% | 60% | 人工终审主导 |
