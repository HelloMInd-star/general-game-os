## 📐 底层逻辑

### 画圈均衡范式
半径 → 供给 · 速度 → 需求 · 方向 → 政策
每一次扰动 = 系统重调
静态圈 = 动态圈的特例（速度 = 0）

text

### 凯利公式 · 量化引擎
f* = (b × p − q) / b

text

- **f***：最优投入比例
- **b**：净赔率
- **p**：胜率
- **q**：1 − p
